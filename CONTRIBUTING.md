# Contributing to [ASYNTH]

Thank you for your interest in contributing to this project! Below are some guidelines to follow to help you get started.

## How to Report Bugs
- Check existing issues to see if the bug has already been reported.
- If not, open a new issue with details on how to reproduce the bug.

## How to Suggest Features
- Open a new issue and use the "Feature Request" template.
- Clearly describe the feature and its benefits.

## Code Contribution Guidelines
- Fork the repository and clone it locally.
- Create a new branch for your feature or bug fix.
- Submit your pull request with a clear description of your changes.
